# Google-Maps-App

### An application for training on Google Maps, in this application there are several screens:

- An activity to get your current location. <br>

- Another activity where you can search for a specific place on the map. <br>

- Another activity where you can change the type of map. <br>

- Another activity for tracking on the map where you can enter your current location and the other place you are going to, then it takes these two places and goes to the Google Map application on your phone, and if the application is not there, it will go to the Play Store to install it.


 ## And the tools used within the application:
- language: Kotlin <br>
- Material Design <br>
- viewBinding <br>
- GoogleMap <br>
- Services location <br>
- Location permission <br>
